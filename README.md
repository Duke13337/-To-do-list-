# To-Do List Project
Простое консольное приложение для управления списком задач на Python.
## Как запустить:
1. Установите Python.
2. Запустите файл командой: `python main.py`

## Запуск через Docker
1. Соберите образ: `docker build -t todo-app .`
2. Запустите: `docker run -it todo-app`

## Запуск через Docker Compose
Выполните команду: `docker-compose up`
